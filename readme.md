# 🔬 AI Research Assistant

A Hierarchical Multi-Agent System for AI/Healthcare Research

## Features
- 4-source search: ArXiv + Semantic Scholar + PubMed + CORE
- Paper type classification (Core Systems vs Applications)
- Citation velocity scoring
- Paper Comparison Engine
- Knowledge Graph visualization
- Research report generation
- BibTeX export

## Setup
Add GROQ_API_KEY in Space secrets settings.

## Built With
LangGraph · LangChain · FAISS · Groq LLM · Gradio
